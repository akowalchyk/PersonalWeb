Seattle Mariners Attendance Predictor

  • Using Python, web-scraped baseball-reference.com for yearly game-by-game statistics of Seattle Mariners home games dating back to 1994 and stored the data in a database using SQL and AWS RDS.

  • Developed and tested machine learning models to include a Random Forrest Regressor and Gradient Boosted Regressor based on exploratory data analysis generated using seaborn.

  • Deployed the best model as a user-friendly Flask app to predict the attendance of a Seattle Mariners home game using HTML and Bootstrap CSS. Launched as a Heroku app to the web.
